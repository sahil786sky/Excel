# Startup Funding Analysis (Excel MIS Project)

## 📌 Overview
This project focuses on analyzing startup funding data using Microsoft Excel. It demonstrates data cleaning, transformation, and reporting techniques used in MIS (Management Information Systems).

---

## 🎯 Objectives
- Clean and structure raw funding data
- Perform data analysis using Excel formulas
- Generate insights on funding trends
- Identify top cities and startups based on funding

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas

---

## 📊 Key Features

### 🔹 Data Cleaning
- Removed null values and inconsistencies
- Standardized city names and formats

### 🔹 Data Analysis
- Applied formulas:
  - VLOOKUP  
  - XLOOKUP  
  - SUMIFS  
  - COUNTIF  
  - IF statements  
- Extracted Year and Month from date column

### 🔹 Reporting & Insights
- Total funding by city
- Number of startups by city
- Funding categorization (High / Medium / Low)
- Identification of top-performing cities and startups

### 🔹 Pivot Tables
- Funding trend by year
- Funding distribution by city
- Top startups by funding amount

---

## 📈 Sample Insights
- Major funding is concentrated in top metro cities
- Funding trends show variation across years
- Few startups receive significantly higher investments

---

## 🚀 Project Structure
- Raw Data → Original dataset  
- Cleaned Data → Processed dataset  
- Analysis → Formulas and calculations  
- Pivot → Summary tables  
- Dashboard → (Optional / In Progress)

---

## 💡 Conclusion
This project demonstrates how Excel can be used as a powerful tool for MIS reporting and business decision-making.
